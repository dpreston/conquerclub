# ConquerClub Overlay

The ConquerClub Overlay is a Chrome extension for the http://conquerclub.com website.

## Prerequisites

You will need the following things properly installed on your computer.

* [Git](http://git-scm.com/)
* [Node.js](http://nodejs.org/) (with NPM)
* [Bower](http://bower.io/)
* [Ember CLI](http://www.ember-cli.com/)
* [PhantomJS](http://phantomjs.org/)

## Installation

* `git clone https://github.com/dpreston/conquerclub.git` 
* `cd conquerclub`
* `npm install`
* `bower install`
